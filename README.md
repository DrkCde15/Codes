Codes

Este repositório contém diversos códigos e projetos desenvolvidos em diferentes linguagens de programação e tecnologias. Cada pasta é dedicada a uma linguagem ou área de estudo, organizada para fácil navegação e consulta.

Estrutura do Repositório
C: Contém exemplos e projetos desenvolvidos em C. Aqui você encontrará implementações de diversos algoritmos e estruturas de dados, bem como exercícios práticos.

HTML, CSS, Javascript: Esta pasta inclui arquivos relacionados ao desenvolvimento web. São exemplos de front-end e back-end, usando HTML, CSS e JavaScript.

Java e JS (POO-PAOO): Códigos e projetos que demonstram a aplicação de Programação Orientada a Objetos (POO) com Java e JavaScript, com foco em arquitetura e design de software.

Python: Scripts em Python voltados para o estudo de estruturas de dados e algoritmos, incluindo implementações de listas e outros tópicos avançados.
